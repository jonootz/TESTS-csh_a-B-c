Editing the Master Page

Article #: article # Product: Product Name Version: Version #

Summary:

Describes how to edit the Master Page, which compiles additional elements onto topic pages in the output. I have a monkey here.

Full Description:

Master Pages can be used to add content to every page in an output. This project uses a Master Page to add breadcrumbs above the topic, and a copyright below the topic in WebHelp output. When editing the Master Page, you will see proxies representing areas of the topic that will be populated at compile time.

Click the images below to see the Master Page source compared to the generated topic.

Source Output

 

Solution:

The Master Page can be found in Content/Resources/MasterPages in the Content Explorer or Added from the menu "Project -> Add Master Pate..."

  1. Double-click to open in the editor
  2. Add, remove, or move proxies and additional content as needed
